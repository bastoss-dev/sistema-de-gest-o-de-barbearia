Sistema de Gestão de Barbearia

Este projeto foi desenvolvido em Python para simular o funcionamento básico de uma barbearia, realizando o atendimento de clientes e o controle dos serviços prestados.

Funcionalidades

Atendimento de clientes em fila
Escolha de um ou mais serviços por cliente
Serviços com valores fixos
Cálculo do valor total arrecadado
Registro da quantidade de serviços realizados

Conceitos Utilizados

Listas
Estruturas de repetição (for e while)
Estruturas condicionais (if e elif)
Variáveis e contadores
Entrada e saída de dados

Objetivo

Aplicar na prática os conceitos básicos de programação estudados em sala de aula por meio da criação de um sistema simples de gestão para uma barbearia.
